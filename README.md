
✍️ # **Handwritten Digit Recognition **

A mini Machine Learning project that classifies handwritten digits (0–9) using the MNIST dataset. This project focuses on learning core computer vision and deep learning concepts through practical implementation.

**a. Key Learnings**

1.Image preprocessing and normalization

2.Neural network training and evaluation

3.Digit classification using MNIST data

**b. Tech Stack**

Python, TensorFlow/Keras, NumPy

**c. Purpose**

Created as a hands-on mini project to strengthen Machine Learning fundamentals and build confidence in computer vision tasks.
